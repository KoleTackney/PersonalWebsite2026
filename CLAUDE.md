# CLAUDE.md

This is koletackney.dev — a personal portfolio/resume website for Kole Tackney, replacing the old Next.js static site.

## Stack

- **Go 1.25** with stdlib `net/http` (no third-party router)
- **Templ** v0.3.x — type-safe HTML components compiled to Go
- **HTMX** — interactivity without a JS framework
- **Tailwind CSS** v4.2.2 — standalone CLI binary (not npm)
- **Module name:** `PersonalWebsite2026`

## Project Layout

```
cmd/api/main.go          — Application entrypoint, graceful shutdown
cmd/web/                  — Templ templates (.templ), handlers, embedded assets
  ├── assets/css/         — Generated Tailwind output (gitignored)
  ├── assets/js/          — HTMX (vendored)
  ├── styles/input.css    — Tailwind input file
  └── efs.go              — go:embed for static assets
internal/server/          — Server config, route registration, CORS middleware
referenceMaterial/        — Old website HTML, resume content (see its README)
```

## Build & Development

- `make build` — runs templ generate → tailwind compile → go build (produces `main.exe`)
- `make run` — `go run cmd/api/main.go`
- `make watch` — live reload via Air (watches .go, .templ, .html files)
- `make test` — `go test ./... -v`

**Generated files (gitignored, never commit):**
- `*_templ.go` — generated by templ from `.templ` sources
- `cmd/web/assets/css/output.css` — generated by tailwind CLI
